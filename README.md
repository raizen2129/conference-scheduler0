# Meeting Scheduler Website

A simple browser-based meeting scheduler with a calendar view.

## Features

- Add meetings
- View meetings on a monthly calendar
- See list of scheduled meetings
- Delete meetings
- Automatically updates calendar
- Saves data using browser localStorage

## Project Structure

index.html - Main webpage  
style.css - Website styling  
script.js - Website functionality  
README.md - Project documentation

## How to Run

1. Download all files
2. Put them in the same folder
3. Open **index.html** in your browser

No installation required.

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
